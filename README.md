## рџ“Њ Project Overview

The **E-Commerce Sales Dashboard** is an interactive business intelligence dashboard developed using **Microsoft Power BI**. The dashboard analyzes e-commerce sales data from two CSV datasets (**Orders.csv** and **Details.csv**) to provide meaningful insights into sales performance, customer behavior, profit trends, and product categories.

This dashboard enables businesses to monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions through interactive visualizations.
